# Bit Heroes Clone - Web Version 🎮

Um RPG 2D inspirado em Bit Heroes, desenvolvido para rodar diretamente no navegador sem necessidade de instalações ou configurações complexas.

## 🚀 Como Jogar

1. Baixe os arquivos do repositório.
2. Navegue até a pasta `/web`.
3. Abra o arquivo `index.html` em qualquer navegador moderno (Chrome, Firefox, Edge, etc.).
4. A batalha começará automaticamente!

## ⚔️ Funcionalidades

- **Combate em Turnos Automático**: O herói e o inimigo se atacam sequencialmente.
- **Sistema de Vida e Dano**: Cálculos de dano baseados em ATK e DEF com variação aleatória.
- **Interface Intuitiva**: Barras de vida dinâmicas e log de combate em tempo real.
- **Pixel Art**: Estilo visual retrô utilizando CSS e sprites leves.
- **Loop de Jogo**: Opção de reiniciar a batalha após a vitória ou derrota.

## 📂 Estrutura do Projeto

- `/web/index.html`: Estrutura principal do jogo.
- `/web/style.css`: Estilização, animações e sprites (Base64).
- `/web/game.js`: Lógica do sistema de combate e interface.
